# sqa_basic.py

**Goal:** Run simulated quantum annealing (SQA) with user-defined beta/gamma schedules.

**What it demonstrates**
- User schedule control
- Trotter slices and replicas

**Run**
```bash
python qanneal/examples/python/sqa_basic.py
```

**Expected output (example)**
```
Best energy: <value>
Energy trace length: <steps>
```
